### 🖼️ Access to images

Each image is available via a direct link:

- **Main** (`courseid = 1`):  
  `https://raw.githubusercontent.com/YesSeir/cs2kz-maps/main/maps/{mapname}.jpg`

- **Bonus** (`courseid > 1`):  
  `https://raw.githubusercontent.com/YesSeir/cs2kz-maps/main/maps/{mapname}_{courseid}.jpg`

- `kz_devon`, Main:

### 📄 Info about maps

File `maps.json` contains full information about each map and its courses. It is available at the following link:

- [https://raw.githubusercontent.com/YesSeir/cs2kz-maps/main/maps.json](https://raw.githubusercontent.com/YesSeir/cs2kz-maps/main/maps.json)

### Structure JSON


```json
{
    "mapname": "kz_devon",
    "coursename": "Main",
    "courseid": 1,
    "workshopid": "3434238689",
    "ckznubtier": 3,
    "ckzprotier": 3,
    "vnlnubtier": 5,
    "vnlprotier": 6,
    "image": "https://raw.githubusercontent.com/YesSeir/cs2kz-maps/main/maps/kz_devon.jpg"
}
```
